# Slack

### Slack message sending

Enabling this action allows your dashboard to send a message to your Slack channel whenever the user mentions any topic that you want to be notified with. 

Check the steps to integrate Slack with Chatbase through this page.

1. Click on the **Create Action** button of Slack in the AI Actions menu:

<Frame>
  <img src="https://mintcdn.com/chatbase/DIWSTyTSabxfVW3B/user-guides/chatbot/images/actions/actions-1.png?fit=max&auto=format&n=DIWSTyTSabxfVW3B&q=85&s=066aeb7b4a9064b005df0a0ad1166c49" alt="Chatbase Embed Code Example" width="896" height="538" data-path="user-guides/chatbot/images/actions/actions-1.png" />
</Frame>

2. Choose the Slack workspace that you want to be connected with this Action:

<Frame>
  <img src="https://mintcdn.com/chatbase/DIWSTyTSabxfVW3B/user-guides/chatbot/images/actions/actions-2.png?fit=max&auto=format&n=DIWSTyTSabxfVW3B&q=85&s=f3c0cdfc3186ecb5cc5e1bab5e4e1a27" alt="Chatbase Embed Code Example" width="599" height="323" data-path="user-guides/chatbot/images/actions/actions-2.png" />
</Frame>

3. In the **When to use** section, provide a detailed description explaining when we should use this action. Include examples of the data this action provides and customer queries it helps answer.

<Frame>
  <img src="https://mintcdn.com/chatbase/DIWSTyTSabxfVW3B/user-guides/chatbot/images/actions/actions-3.png?fit=max&auto=format&n=DIWSTyTSabxfVW3B&q=85&s=3fd9d1001247d0de1b613023b5bc0660" alt="Chatbase Embed Code Example" width="1572" height="600" data-path="user-guides/chatbot/images/actions/actions-3.png" />
</Frame>

4. Click on Save button.

5. Make sure to enable this action to allow us to send a message to your Slack channel.

<Frame>
  <img src="https://mintcdn.com/chatbase/DIWSTyTSabxfVW3B/user-guides/chatbot/images/actions/actions-4.png?fit=max&auto=format&n=DIWSTyTSabxfVW3B&q=85&s=99ea878ee3197bfd683ecd99cfc7856d" alt="Chatbase Embed Code Example" width="1600" height="300" data-path="user-guides/chatbot/images/actions/actions-4.png" />
</Frame>

Now, you can try this action in the Playground on the right side of the page. It's recommended to try sending a message in the Action preview to ensure that the AI agent sends a message to your Slack channel when the desired instructions are fulfilled before enabling the action.

Examples:

"Call this tool to send a message in slack to the channel named: ai-actions-slack whenever the user mentions any of the following topics: standard plan"
