# Frontend Integration

Frontend-facing Chatbase documentation for embed widgets, custom browser integrations, UI building blocks, and platform-specific embed guides.

## Sections

- [`embed-widget/`](embed-widget/) - JavaScript embed, widget control, identity verification, suggested/initial messages, and user conversation continuity.
- [`custom-frontend/`](custom-frontend/) - Client actions, event listeners, widgets/components, forms, layout, media, and display primitives.
- [`platform-embeds/`](platform-embeds/) - Low-code/CMS embed guides for Bubble, Framer, Webflow, Weebly, Wix, and WordPress.
